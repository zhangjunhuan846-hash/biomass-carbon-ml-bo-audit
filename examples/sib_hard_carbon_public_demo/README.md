# SIB hard carbon public synthetic demo dataset

This dataset is **synthetic**. It is intended only to test the carbon-literature BO replay workflow.

It does not contain real DOI, article titles, author names, or original sample identities from the user's unpublished review database.

Use it to test:

- missingness audit
- paper-level leakage checks
- protocol completeness checks
- quality-tier distribution
- source-PDF recheck queue generation
- offline BO replay mechanics

Do not use this dataset for scientific conclusions.
